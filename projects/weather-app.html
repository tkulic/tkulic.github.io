WeatherApp

local weather data

Overwiew

This application represents an enhanced solution to the freeCodeCamp Challenge "Show the local weather". Its main functionality is displaying weather data for the current location.

Technologies

  • HTML5
  • CSS3
  • JavaScript (ES6)
  • Geolocation API
  • freeCodeCamp Weather API

Fulfilled user stories:

  • I can see the weather in my current location.
  • I can see a different icon or background image (e.g. snowy mountain, hot desert) depending on the weather.
  • I can push a button to toggle between Fahrenheit and Celsius.
  • I can switch between dark and light mode. (Bonus feature not included in the challenge)
